Who I Am

I'm a Computer Science M.S. student at Missouri University of Science and Technology (Missouri S&T), where my research focuses on AI-driven frameworks for crack width measurement in structural health monitoring imagery. My thesis develops annotation pipelines, segmentation models, and width estimation algorithms to automate structural inspection.

Previously, I interned at the USGS National Geospatial Technical Operations Center, co-authoring peer-reviewed work on deep learning transfer for hydrographic feature extraction and cartographic generalization. I also spent a semester as a UF Pathfinder Fellow working on 3D reconstruction and autonomous navigation for greenhouse robotics at the University of Florida.

Outside of research I co-founded REDACTED, an agricultural robotics startup building autonomous pollination systems for controlled-environment agriculture.

Embedded Climate Controller (PID vs. Fuzzy-PID)
GitHub

Built a closed-loop environmental chamber on a Raspberry Pi 4 (ABE5009 — Control Systems in SmartAg). Implemented both standard PID and a fuzzy gain-scheduled PID to control a MOSFET-switched heating pad and fan inside a foam cooler. Modeled the thermal plant in Simulink and compared controllers experimentally: fuzzy-PID achieved 3.6× faster rise time (51 s vs 187 s) but PID achieved tighter steady-state regulation (±0.17 °C error vs −0.47 °C).

Raspberry Pi PID / Fuzzy-PID Python Simulink DHT11 Embedded Control
XBee Wireless Sensor Network
GitHub

Designed and deployed a multi-node wireless sensor network (ABE6031) using XBee radios, an Arduino Uno (Node 1: DHT11 + SGP30 eCO₂/TVOC + photoresistor), and an Elegoo MEGA 2560 (Node 2: DHT11 + photoresistor). A Raspberry Pi 4 gateway serialized all sensor CSVs, uploaded 8-channel telemetry to ThingSpeak every 20 s, and pushed Pi Camera images to Google Drive via rclone every 60 s.

XBee Arduino Raspberry Pi SGP30 eCO₂/TVOC ThingSpeak Python IoT
LabVIEW Signal Processing Pipeline

Developed a five-stage sensor signal processing pipeline in LabVIEW (ABE6031 Take-Home Final). Stages: (1) Z-score outlier removal (σ = 2 threshold, replaced with prior value), (2) IIR Butterworth lowpass filter at 400 Hz cutoff, (3) FFT spectral analysis with Hanning window and Peak Magnitude measurement, (4) Fourier series reconstruction at 4 dominant frequencies (~100, 200, 300, 401 Hz), and (5) RMS error = 4.84 between filtered and reconstructed signals.

LabVIEW Signal Processing FFT / Fourier Butterworth Filter Instrumentation
Greenhouse Robotics (UF Pathfinder)

DNABERT-2 Fine-tuning for DNA Error Correction
GitHub

Co-authored graduate ML project (CS6001 — Missouri S&T, w/ Luke Hendrickson) fine-tuning DNABERT-2 for per-base DNA sequencing error detection. Synthesized 3.2 M 150 bp reads from chr21 T2T reference with 10 % indel + 1 % substitution noise. Replaced softmax with sigmoid multi-label output (dim = 150), applied LoRA, and fixed a Triton Flash Attention matrix-transpose bug. Trained on NVIDIA A100 80 GB. F1: 0.651 → 0.995; Recall: 0.486 → 1.0.

PyTorch DNABERT-2 LoRA Flash Attention CUDA / A100 Bioinformatics
